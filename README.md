# Radoskop

Open source tool for monitoring voting sessions in Polish city councils (Rada Miasta). Parses PDF protocols from BIP (generated by the eSesja system) and builds dashboards with per-councillor metrics: attendance, voting activity, party loyalty, rebellious votes, and voting similarity matrices.

Currently monitoring **13 cities**, **457 councillors**, **304 sessions**, **10,406 individual votes**, and **16,725 interpellations**.

Live dashboards: [radoskop.pl](https://radoskop.pl)

## Monitored cities

Warszawa, Kraków, Łódź, Wrocław, Poznań, Gdańsk, Szczecin, Bydgoszcz, Lublin, Białystok, Katowice, Gdynia, Sopot.

## Quick start

```bash
pip install pdfplumber

# 1. Download PDF protocols from your city's BIP
# e.g. https://bip.gdansk.pl/rada-miasta-gdanska/Protokoly-z-sesji-Rady-Miasta-Gdanska,a,771

# 2. Parse PDFs
python scripts/parse_pdf.py pdfs/ --out data/

# 3. Build metrics and dashboard data
python scripts/build_metrics.py data/ --out dashboard/data.json

# 4. Run dashboard
cd dashboard && python -m http.server 8000
# Open http://localhost:8000
```

## What it measures

- **Frekwencja** — % of sessions the councillor attended
- **Aktywność** — % of votes where the councillor actually voted (for/against/abstain vs absent/no vote)
- **Zgodność z klubem** — % of votes matching the majority of the councillor's party group
- **Buntownicy** — votes where the councillor voted against their party group majority
- **Macierz podobieństwa** — who votes most similarly to whom

## Scaling to other cities

The parser works with any PDF generated by eSesja (used by hundreds of Polish municipalities). To add a new city, download the PDF protocols from their BIP page.

## Use of Generative AI

This project uses generative AI tools (Claude, GitHub Copilot) during development for code generation, data analysis, and documentation. All AI-generated outputs are reviewed and validated by the maintainer before inclusion.

## Data source

Session protocols published in BIP (Biuletyn Informacji Publicznej) by each city council. Format: PDF generated by the eSesja system with roll-call votes (mandatory since 2018 by Polish law).
